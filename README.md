* Click & Book

Aplicación web desarrollada en Flask como prototipo funcional para la Segunda Entrega.

* Alumnas:

-Monla, Deborah
-Michienzi, Agustina
-Gimenez, Paula

* Funcionalidades:

- Registro e inicio de sesión de usuarios.
- Gestión de reservas (crear, consultar, modificar y cancelar).
- Catálogo de servicios fotográficos.
- Predicción mediante Inteligencia Artificial.
- Despliegue en Render con PostgreSQL.

* Instalación:

```bash
pip install -r requirements.txt
python run.py
```
